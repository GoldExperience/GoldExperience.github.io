ROS IDE ：Pycharm2018.3+ROS-melodic

P O R T
2020-09-07 (Updated: 2020-09-07)

第一次安装使用pycharm编写ros程序的时候会出现无法import rospy这种事情，下面是解决方案
第一步
打开setting -》Project Interpreter 查看当前环境

看看是不是上图所示的Project interpreter
安装好ros-melodic之后在终端输入 python ，可以看到当前环境下Python的版本，按ctrl-D退出Python

第二步，如果不是，选择“show all”查看是否存在。
如果不存在，就选择第三个System Interpreter，找到当前用的Python2的环境

第三步，查看当前的Project interpreter，发现已经有rospkg,rosinstall等包了，但是没有rospy

第四步，选择apply，ok，右下角会显示正在处理，等待一段时间处理完之后，可能在import rospy的时候还是会出现红色波浪线，这是因为我们没有source setup.bash的原因。

第五步，暂时不想修改pycharm的desktop文件，想修改的话参照这个pycharm setup
简单的方法是不要从dash启动pycharm，而是从terminal启动，因为我们已经在bashrc里source了setup.bash，在terminal内输入pycharm-community启动pycharm

如果bashrc里并没有source的话就手动source一下
